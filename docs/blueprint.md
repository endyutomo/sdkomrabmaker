# **App Name**: BOQ Builder

## Core Features:

- AI-Powered Item Suggestion: Suggest relevant items for the Bill of Quantities based on the project type and specifications using a generative AI tool. It also categorizes each item under relevant groups to improve organisation.
- Template Import: Import existing BOQ templates in common formats (e.g., Excel, CSV) to quickly populate the bill.
- Manual Item Entry: Allows users to manually add and edit items with descriptions, quantities, units, and costs.
- Cost Calculation: Automatically calculate costs based on quantity and unit price. The tool will summarize costs per section.
- BOQ Preview and Export: Preview the Bill of Quantities and export it into PDF, Excel, or other specified formats.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to evoke trust and professionalism.
- Background color: Light gray (#E9E9F1) for a clean, neutral backdrop.
- Accent color: Electric purple (#BE0AFF) to highlight important actions and information.
- Font: 'Inter', a grotesque-style sans-serif font, will be used for a modern and professional appearance for both headings and body text.
- Use a set of professional icons to represent various BOQ components and functions.
- A clean and structured layout with clear sections and subsections for ease of navigation.
- Subtle animations for feedback on interactions (e.g., item addition, saving).
